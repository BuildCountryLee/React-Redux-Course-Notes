## 第2章 H5开发

### getParamUrl

用正则实现获取浏览器地址栏参数

```javascript
// utils.js
export default {
  getUrlParam(name) {
    let reg = new RegExp('(^|&)'+name+'=([^&]*)(&|$)');
    let r = window.location.search.substr(1).match(reg);
    if(r !=null) return decodeURIComponent(r[2]);
  }
}
```
